Typography

Standard paragraph text uses var(--fg-secondary) for color. You can use bold text for emphasis, and inline code for technical terms. Links should look like this.

Heading Level 3

Use headings to create a clear hierarchy. H2 for major sections, H3 for subsections, H4 for details.

Code Blocks

Use code blocks for terminal commands, configuration files, and code examples.

Basic Code Block

ato install my-package

# This is a comment
Output from the command

Configuration File Example

# ato.toml

[package]
name = "my-package"
version = "1.0.0"
description = "A sample package"

[capsule]
type = "cli"
entry = "./src/index.ts"

Syntax Highlighting Classes

# comment - Gray comments
command      # Green commands
[section]    # Blue package/section names
key =         # Pink keys
"value"      # Yellow strings
--flag       # Purple flags/options
Error:       # Red error messages
Output text  # Gray output
